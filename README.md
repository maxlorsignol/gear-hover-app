Gear Hover App

A lightweight interactive web app that displays a hand-drawn gear layout in greyscale and reveals each item in full color on interaction. Clicking an item opens a short descriptive note.
Built with plain HTML, CSS, and JavaScript using a <canvas>-based image overlay technique.
